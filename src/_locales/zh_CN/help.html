帮助
添加搜索引擎
在任意网页的搜索框上单击右键并选择“添加到 ContextSearch”，也可在搜索引擎管理器中手动添加新引擎。


网页右键菜单
打开方式
划选字词后或直接在网页内单击右键打开网页右键菜单，展开名为“搜索 选中内容（或页面等）”的 ContextSearch 菜单，并选择要使用的搜索引擎。


快捷菜单
打开方式
划选字词并按住右键直到出现快捷菜单。你可以从多个选项中选择打开快捷菜单的方式。


调整大小和重新排列
快捷菜单可以通过改变“列数”和“行数”设置，或打开菜单布局编辑器拖拽边框右下角来调整界面大小。

以下两种方法可重新排列搜索引擎，并同时改变搜索引擎管理器中的引擎顺序。

1、勾选高级选项 alwaysAllowTileRearranging 后，可直接在菜单中拖放引擎图标进行排序；
2、打开菜单工具栏中的布局编辑器，拖放引擎图标进行排序。


工具栏
打开方式
使用 ContextSearch 的工具栏按钮可随时访问引擎。


搜索引擎管理器
添加文件夹
在文件夹和子文件夹中组织整理搜索引擎。

1、在搜索引擎管理器中单击右键菜单并选择“新建文件夹”；
2、以唯一的名称命名该文件夹；
3、拖放搜索引擎到文件夹中。


编辑图标
使用自定义搜索引擎图标。

1、双击搜索引擎或右键单击搜索引擎选择“编辑”；
2、在“图标 URI”处填入新图标链接并点击“保存”。


导入火狐搜索引擎
自 Firefox 63 开始，一键搜索引擎将自动导入 ContextSearch，但除非这些引擎是通过文件资源管理器导入的，否则将无法修改。

如果你打算修改一键搜索引擎，可点击搜索引擎页面“导入一键搜索引擎”下方的按钮导入，浏览配置文件夹并选择 search.json.mozlz4 文件。你可以在地址栏输入 about:profiles 后按回车打开来找到配置文件路径。

  一般的路径：

  Windows

%APPDATA%\Mozilla\Firefox\Profiles\<配置目录名称>\search.json.mozlz4

  Mac OS X

Users/<username>/Library/Application Support/Firefox/Profiles/<配置目录名称>/search.json.mozlz4

  Linux

/home/<username>/.mozilla/firefox/<配置目录名称>/search.json.mozlz4
